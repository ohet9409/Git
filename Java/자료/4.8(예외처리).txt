* 예외처리 
=>  실행중에 에러가 발생했을 때, 프로그램이 멈추는 것을 방지하는 것
try {
	에러가 발생할 만한 명령어
} catch(Exception e) {		--> catch 여러개 사용 가능
	에러가 발생했을 때 실행할 명령어
} finally {
	에러가 생기든, 생기지 않든 실행할 명려어
}

* 예외처리 방법
1. try-catch 사용
2. 예외 전가
=> 클래스를 만들때, 에러가 발생할 것 같은 코드에 직접 예외처리를 하는
것이 아니라, 이 클래스를 사용하는 곳에서 예외처리를 하도록 미루는 것

* 파일에 데이터 저장하기
1. 쓰기
   RAM         ->        HDD
           변환       출력
   문자열 -> byte[]  -> 파일
2. 읽기
   HDD         ->         RAM
                입력                변환
   파일         ->         byte[]  -> 문자열
* 파일에 저장하는 형태
1. 문자 저장
 -> text file
 -> "abc" uni code 형식 저장(2byte) => 0097 0098 0099
 -> "abc" utf-8 형식 저장(한글 3byte, 영어 1byte) => 97 98 99
     "가" -> ea b0 80
2. 데이터 저장
 -> data file
 -> data format이 정해져 있음
 -> ~.bmp
     1px = rgb(ffffff)
